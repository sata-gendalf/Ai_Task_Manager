# AI Task Assistant

AI Task Assistant — система управления задачами с автоматическим анализом текста задач.

## Структура проекта

- backend — Node.js + Express API
- frontend — React-приложение
- python-service — Python-сервис для анализа задач
- database — SQL-скрипты для PostgreSQL

## Основной функционал

- регистрация пользователя
- вход в систему
- JWT-авторизация
- создание задач
- просмотр только своих задач
- автоматическое определение приоритета и категории задачи
